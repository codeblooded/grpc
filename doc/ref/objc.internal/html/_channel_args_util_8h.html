grpc_channel_args* GRPCBuildChannelArgs ( NSDictionary *  dictionary)

Allocates a grpc_channel_args and populates it with the options specified in the dictionary.

Keys must be NSString, NSNumber, or a pointer. If the value responds to @selector (UTF8String) then it will be mapped to GRPC_ARG_STRING. If the value responds to @selector (intValue), it will be mapped to GRPC_ARG_INTEGER. Otherwise, if the value is not nil, it is mapped as a pointer. The caller of this function is responsible for calling GRPCFreeChannelArgs to free the grpc_channel_args struct.

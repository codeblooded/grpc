32 // The main purpose of code here is to provide means to cache SSL sessions
33 // in a way that they can be shared between connections.
34 //
35 // SSL_SESSION stands for single instance of session and is not generally safe
36 // to share between SSL contexts with different lifetimes. It happens because
37 // not all SSL implementations guarantee immutability of SSL_SESSION object.
38 // See SSL_SESSION documentation in BoringSSL and OpenSSL for more details.
39
